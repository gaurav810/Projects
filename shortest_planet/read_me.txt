
Read Me 
==========

Used Technology

1. Spring Boot
2. Hibernet
3. Maven
3. MySQL
4. Angular JS


1. Create Database in MySQL "assignment"

CREATE SCHEMA `assignment`;

2. Execute below command for build project

Project location > mvn clean install

3. Start Server

Project location > mvn spring-boot:run

4. Access Project

localhost:8080


Please find images for front end in case folder.
And Import assignment project in eclipes.
